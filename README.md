## Environment variables
```shell
set FLASK_ENV=development
```
```bash
set FLASK_APP=todo
```

```bash
set FLASK_DATABASE_HOST='127.0.0.1'
```
```bash
set FLASK_DATABASE_PASSWORD=mypwd
```
```bash
set FLASK_DATABASE_USER=myuser
```
```bash
set FLASK_DATABASE=mydb
```

Suscribiendo comando
```bash
flask init-db
```
